This is EVERYTHING I've learned about Reolink, their NVR system, PoE, and Solar Cameras. 

Reolink NVR Kits - This is great if you're trying to save money and have a simple PoE plug and play solution. This does add difficulties to manage as all the streams are under a single device and kit camera NEED the NVR to function. Some kit cameras may have more features so check the specific features of the camera model in the kit. You can probably find a great deal used.
REOLINK 8CH 5MP: https://amzn.to/4dulGgG
REOLINK 16CH 12MP: https://amzn.to/4d9YaWz (my kit)

Reolink PoE Cameras - I've had no issues with these. Great for the Reolink ecosystem and using 3rd party services such as Frigate.
12MP UHD PoE: https://amzn.to/4d6BuGJ
4K Floodlight: https://amzn.to/4ceRKnM
