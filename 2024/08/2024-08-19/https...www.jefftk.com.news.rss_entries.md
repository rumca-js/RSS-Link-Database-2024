<p><span>

I maintain a </span>

<a href="https://www.jefftk.com/apartment_prices/">Boston
apartment price map</a>, scraping 

<a href="https://www.padmapper.com/">Padmapper</a>'s listings to make a
heatmap. Each month I run a script to update it, but last month it
failed due to an API change.  It had been impressively stable: I'd
been running the same code unmodified for 

<a href="https://github.com/jeffkaufman/apartment_prices/commits/master/query_padmapper.py">three
years</a>, with only minor modifications since a big rewrite in
2016.



<p>
