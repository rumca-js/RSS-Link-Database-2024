## ValleyRAT malware is targeting Chinese-speaking users
 - [https://securityaffairs.com/167164/cyber-crime/valleyrat-malware-targets-chinese-speaking-users.html](https://securityaffairs.com/167164/cyber-crime/valleyrat-malware-targets-chinese-speaking-users.html)

 - date published: 2024-08-17T07:15:41+00:00

FortiGuard Labs researchers uncovered an ongoing ValleyRAT malware campaign that is targeting Chinese-speaking users. ValleyRAT is a multi-stage malware that supports multiple techniques to monitor and control compromised devices. The malicious code is also used to deploy arbitrary plugins on the infected systems. A noteworthy characteristic of ValleyRAT malware is the heavy usage of shellcode [&#8230;]
