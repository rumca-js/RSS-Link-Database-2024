Lightning Talk: Single Responsibility Principle (SRP) for Function Calls in C++ - Stewart Becker - ACCU 2024
---

Calling a function means doing some work AND getting a value back.  There is a class in the C++ standard library that separates these two aspects, but it doesn't get enough love.  This talk explores what is possible with this class, and a much-needed extension.
