<p>Unified endpoint management (UEM) describes a set of technologies used to secure and manage a wide range of employee devices and operating systems — all from a single console.</p>



<p>Seen as the next generation of mobility software, UEM tools incorporate several existing <a href="https://www.computerworld.com/article/1713505/what-is-enterprise-mobility-management-emm.html">enterprise mobility management (EMM)</a> technologies — including <a href="https://www.computerworld.com/article/1710837/whats-the-difference-between-mdm-mam-emm-and-uem.html">mobile device management (MDM) and mobile application management (MAM)</a> — with some of the tools used to secure desktop PCs and laptops.</p>
