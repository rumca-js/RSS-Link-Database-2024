## New Raspberry Pi OS Release Brings Many UI Improvements and New Settings
 - [https://www.linuxtoday.com/news/new-raspberry-pi-os-release-brings-many-ui-improvements-and-new-settings](https://www.linuxtoday.com/news/new-raspberry-pi-os-release-brings-many-ui-improvements-and-new-settings)
 - RSS feed: https://www.linuxtoday.com/news/feed
 - date published: 2024-07-05T13:00:52+00:00

<p>The new Raspberry Pi OS release, versioned 2024-07-04, updates the panel and the PCManFM file manager with support for customizing more than two desktops. The panel also received support for customizing the labwc Wayland window-stacking compositor, which is now installed by default as an alternative to Wayfire.</p>
