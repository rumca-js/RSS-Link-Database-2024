 - RSS feed: https://hnrss.org/frontpage
 - date published: 2024-02-02T18:48:53+00:00

<p>I'm experimenting game mechanics that are usually seen on a typical grid, but transposed on an irregular one.<p>This is a first game based on Minesweeper.<p>The changes an irregular grid brings are that cells don't all have 8 neighbors. Some can have less, some can have more. So when you try and deduce if a tile has a mine, you have to pay closer attention at the number clues and which tiles neighbors it.<p>My intention is to make a very classic game a bit more challenging & fun.<p>I've also released a first version of Checkers/Draughts, and plan to test similar variants for Chess, Othello and Go.</p>
