## Pallets projects added to scope of PSF CVE Numbering Authority
 - [https://pyfound.blogspot.com/2024/08/pallets-projects-now-in-scope-for-psf-cna.html](https://pyfound.blogspot.com/2024/08/pallets-projects-now-in-scope-for-psf-cna.html)
 - RSS feed: https://pyfound.blogspot.com/feeds/posts/default?alt=rss

<div style="text-align: left;">Last year the Python Software Foundation was <a href="https://pyfound.blogspot.com/2023/08/psf-authorized-as-cna.html">announced as a CVE Numbering Authority</a> (CNA) to manage and assign CVE IDs for CPython and pip. Becoming a CVE Numbering Authority allows the PSF to provide expertise about Python in the CVE ecosystem, ensuring that users have accurate and up-to-date information about vulnerabilities affecting key projects.</div><div style="text-align: left;"><br /></div><div style="text-align: left;">Today, the PSF is <a href="https://www.python.org/cve-numbering-authority/">expanding our CNA scope</a> to also include Pallets projects, such as <a href="https://github.com/pallets/flask">Flask</a>, <a href="https://github.com/pallets/jinja">Jinja</a>, <a href="https://github.com/pallets/click">Click</a>, and <a href="https://github.com/pallets/quart">Quart</a>.
