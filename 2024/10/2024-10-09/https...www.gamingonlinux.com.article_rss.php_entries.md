## FEX-Emu version 2410 out bringing some important fixes for gaming on AArch64
 - [https://www.gamingonlinux.com/2024/10/fex-emu-version-2410-out-bringing-some-important-fixes-for-gaming-on-aarch64](https://www.gamingonlinux.com/2024/10/fex-emu-version-2410-out-bringing-some-important-fixes-for-gaming-on-aarch64)
 - RSS feed: $source
 - date published: 2024-10-09T09:08:56+00:00

FEX-Emu is an emulation layer that allows you to run x86 and x86-64 binaries (like games and apps) on an AArch64 Linux host, similar to qemu-user and box86/box64. Instead of using full disk images, FEX-Emu uses rootfs overlay + forwarding GL processing to the host to achieve better performance in emulated software.<p><img src="https://www.gamingonlinux.com/uploads/articles/tagline_images/268067920id25419gol.jpg" alt />.</p><p>Read the full article on <a href="https://www.gamingonlinux.com/2024/10/fex-emu-version-2410-out-bringing-some-important-fixes-for-gaming-on-aarch64/">GamingOnLinux</a>.</p>
